Distant Lights is an Unreal Engine plugin that creates proxy lights to dramatically increase light draw distance and optimize dynamic shadow distance. Its world subsystem allows managers to act as world lighting managers, blending all distant and actual light intensities quickly and efficiently. Supports Point and Spot lights.


### [🎬Overview](https://youtu.be/PrJWDIklK-w)   |   [🕹️Demo](https://drive.google.com/file/d/11aKZMHRnYBO01FIex4XhtKFRTlOQ28tN/view?usp=sharing)  |   [💡Example Project](https://drive.google.com/file/d/1VhL-Hwu3bqtEKo0B3Q845btprJxpBOEk/view?usp=sharing) 


### Key Features

- **Extended Draw Distance:** Significantly increase light draw distance.
- **Optimized Shadows:** Improves performance by reducing the draw distance of real shadow-casting lights.
- **Depth Fog Support:** Adds simple depth fog for atmospheric, foggy light effects.
- **Animated Flares:** Adds animated flare effects to light components for enhanced visuals.
- **Instanced Light Meshes:** Light meshes are all batched. 
- **World Partition:** Fully supports Unreal's World Partition system.
- **High Performance:** Written entirely in C++.
- **Multiplayer Ready:** Full network replication support.
- **Blueprint:** Friendly for developers who prefer Blueprints.
- **Low Storage Footprint:** Tiny plugin size.
